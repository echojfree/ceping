# 产品需求文档 (PRD): “职境纪元” (CareerVerse) - 大理财贸电商专业专属进化舱

## 1. 产品概述与核心定位
**产品名称**：“职境纪元” (CareerVerse) - 电子商务实战版
**专属客户**：大理财贸职业院校
**目标用户**：大理财贸电子商务专业及相关经管类学生。
**硬件载体**：55寸/65寸 4K 触控“太空舱”式一体机。
**核心理念**：将经典的**霍兰德职业兴趣理论 (RIASEC)** 深度暗埋于“剧本杀”互动中，结合大理财贸电商专业的 6 大核心就业岗位（**产品开发、视觉设计、客户服务、门店经营、电商运营、数据分析**），打造一套“好玩、硬核、且极具指导意义”的实战型职业锚测定与技能养成系统。

## 2. 核心理论架构：霍兰德 (RIASEC) x 电商 6 大岗位
不再直接问学生“你喜欢画画还是算数”，而是通过剧本杀中的**行为选择**反推霍兰德代码，并映射到具体岗位：
*   **R (实用型) / S (社会型)** -> **客户服务、门店经营**：侧重与人沟通、处理实际纠纷、安抚情绪、现场执行。
*   **E (企业型) / C (常规型)** -> **电商运营**：侧重统筹资源、制定促销规则、目标导向、跟进执行流程。
*   **I (研究型) / C (常规型)** -> **数据分析、产品开发**：侧重研究市场趋势、分析竞品数据、核算 ROI 与转化率逻辑。
*   **A (艺术型) / I (研究型)** -> **视觉设计**：侧重页面排版、创新视觉表现、提升点击率 (CTR) 的美学设计。

## 3. 核心交互模块与玩法更新

### 3.1 大理财贸专属赛博枢纽 (The Hub)
*   地图分为三大实战街区，对应电商的不同环节：
    1.  **创意工坊区 (产品/视觉)**：适合 A、I 型潜质。
    2.  **流量中枢塔 (运营/数据)**：适合 E、C、I 型潜质。
    3.  **前线交易所 (客服/门店)**：适合 S、R 型潜质。

### 3.2 极限大促剧本杀 (The Trial: "Double 11 Crisis")
*   **剧情设定**：双十一/618 巅峰之夜，作为店铺操盘团队核心成员，遭遇突发危机（如：头部主播直播间爆款商品超卖 5000 单，供应链断裂，后台差评如潮）。
*   **交互机制**：玩家面对暴躁的“主播选品经理 (NPC)”，每一次选择都代表一种**岗位思维倾向 (Holland 代码倾向)**：
    *   *选择计算赔付率和拉取库存明细* -> 记录为 **C/I 型 (数据/产品倾向)**
    *   *选择优先安抚主播情绪并拟定话术* -> 记录为 **S 型 (客服倾向)**
    *   *选择紧急协调新供应链并调权止损* -> 记录为 **E 型 (运营倾向)**
*   **压力系统**：选择不当会导致店铺 DSR 评分暴跌（替代 HP 条），玩家自身压力值爆表。

### 3.3 霍兰德潜能结算台 (The Evolution Screen)
*   **六边形战士雷达图**：结算页展示霍兰德六角模型 (RIASEC) 的测算结果。
*   **岗位定级**：基于大理财贸的 6 大培养方向，精准输出契合度（例如：“高度契合：电商数据分析师 / 运营专员”）。
*   **电商实战技能树**：扫描二维码，领取针对性的提升任务（例如：“你的【视觉设计】潜能极高，领取现实任务：用 Figma 临摹一张高点击率主图”）。

## 4. UI/UX 视觉风格
*   保留“职境纪元”极具吸引力的 Persona 5 (P5) 游戏化风格、大色块切割与赛博故障风。
*   融入电商元素（如跳动的 GMV 数据面板、转化率走势图、直播间弹幕特效作为背景点缀）。
*   增加大理财贸专属定制的启动徽章与欢迎语，提升学生的归属感和自豪感。
